Welcome to my page for Pokemon First Edition

When I was learning about the creation of the first Pokemon games I read that the creator had originally made a 
single game but a colleague suggested that he split the game into two versions so that the players would have to 
trade to complete the pokedex. Genius. Except now we can't complete pokedexes. 

I had often wondered what that first Pokemon game would have been like, being able to complete the Pokedex in a
solo playthrough, but sadly it was never finshed or released.
So I built it myself.

Following a few rules, I set out to make what I thought of as "Pokemon if the version split never happened" 
and it needed to stay as close to 100% authentic as possible. This meant a few things had to be established.
If something needed to be added to the game, I have to do it either the way Nintendo already did, or how I
think they would've had it never been done in another game of theirs.

So what needed to change? Well the version split is the main thing preventing you from completing the Pokedex, 
because some pokemon are available only in Red or in Blue, like Scyther & Pinsir, but other things were done 
to prevent you from completing the game solo regardless of your version. The total amount of unobtainable
Pokemon are actually these...

- Version Exclusives (Only found in either Red or Blue)
- Choice Pokemon (Starters, fossils, dojo, eeveelutions)
- Trade Evolutions (Only evolves when traded such as Machoke > Machamp)
- Event Pokemon (Mew was only available as a Toys r Us store event in 1999)

Luckily the Gen 1 games actually solve most of these problems for us like this...

- Version exclusives are easy, just merge both encounter and fishing tables right? sorta. Now every area contains
  a possible 12 encounters instead of 10, but where you used to find Oddish OR Bellsprout, now you'll find both.
- Game corner prizes (Scyther OR Pinsir in Red and Blue) was solved in Pokemon Yellow where they remove the
  Nidorans and put both bug Pokemon as prizes, so we did that.
- Starter Pokemon choice was also solved in Yellow where NPCs will give you one of each starter throughout
  the game, so we did that too. The NPCs are mostly the same aside from sprite differences in RBY.
- Eevelutions were handled in Let's Go Pikachu/Eevee, where you can find wild Eeevees on Route 17, so we added
  them there with a low 1% encounter rate.
- The fossil Pokemon is chosen at Mt Moon and then the nerd takes the other one and stays there forever. I assumed
  he eventually finds another one he doesnt need, so after beating the Elite Four you can go find the other fossil
  back in the same location.
- Similarly the Dojo fighting Pokemon you didn't choose stays there forever. Since the leader gave you one for
  defeating him, I assumed he would be equally thrilled by your league victory. So head back there after the E4.
- Trade pokemon were (almost) all handled between Pokemon Yellow and the Japanese Pokemon Blue. Yellow has a
  Machoke trade named "Ricky" which replaces one of the trades in Red/Blue, and it evolves upon trade.
  Japanese Blue also replaces two existing trades with one for a Haunter and one for a Graveler. Fun fact, this
  is why those two traders both say "My Pokemon went and evolved" because they originally did, and now they do again.
  For Alakazam, no Gen 1 game has ever done it so I had to create my own. I did the most Nintendo thing and
  asked the fans. Someone suggested a late game trader requesting a Dragonite so I added in his request.
- One of those trades we replaced was for a Jynx, and that was the only way to obtain one. Now what? JP Blue has us 
  covered again, in their version they made Jynx into a rare wild encounter at the Seafoam Islands, so we did that.
- Lastly there's Mew. There are 3 ways to obtain mew, and only one counts. You can either use a cheat code, use
  a glitch, or attend an event at a Toys r Us in 1999. That last one is the only "real" Mew, so we do that.
  Kanto may not have a Toys r Us, but they do have a massive Department Store in Celadon City. This event was
  rare and the pokemon was mythical, so it's a (very) late game event. After completing the Pokedex and getting
  your Diploma, head back to the Celadon Department Store and look around for a Gentleman you haven't seen before.

... That's all of the Pokemon. By adding all of these in, you can now finally complete the Pokedex.

All that remained was a rebrand of the visuals because Pokemon Red and Blue looked different when played on the
Gameboy Color or Super Game Boy. For the colors, we will use the updated color palette from Pokemon Yellow. 
Now instead of the game being all red or all blue, the color changes based on the town you're in.
The super game boy adds either a red or blue border, so we changed the border to be a new grey one. Lastly the
title screen now displays First Edition and the first Pokemon shown is Rhydon, often called the first Pokemon.
In red/blue there are 16 pokemon that can appear on the title screen and 13 of them are different depending
on your version. In this final version we combine all of them and add one more, making 30 total Pokemon.

I have included 2 releases since I couldnt decide which one I liked better but I knew some people would want both.
The first release is version 1.0 which is exactly what I described above. The second release is version 1.1 which
is the same game but with added content and quality of life features. The changes are described in the release notes 
on each page. All added content is also features that existed in remakes of the Gen 1 games, no new gimmicks.

I do not provide a ROM for this game, or for any games. I do provide a patch which can be applied to your
own copy of Pokemon Red which will turn it into a Pokemon First Edition ROM.

*Note: Save files won't work across versions due to the changes made to the movement engine in v1.1
